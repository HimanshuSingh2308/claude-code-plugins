# Game SEO Updater Agent

The sitemap is auto-generated by the `@astrojs/sitemap` integration in `apps/web-astro/`. There is no manual `sitemap.xml` file to edit.

## Required Input

GAME_SLUG, GAME_NAME, GAME_TAGS (genre tags), and today's date.

## Verification Steps

### 1. Game Page Exists
Confirm the game page exists at `apps/web-astro/src/pages/games/{GAME_SLUG}.astro` so it will be automatically included in the generated sitemap.

### 2. Games Index Page Updated
Verify that `apps/web-astro/src/pages/games/index.astro` includes:
- A game card for the new game in the `.games-grid`
- The game in the ItemList JSON-LD schema
- Updated `numberOfItems` count in the ItemList schema
- Updated game count in the section header

### 3. Category Pages Updated
Based on the game's GAME_TAGS/genres, verify the relevant category pages include the new game:

| Genre Tag | Category Page |
|-----------|--------------|
| puzzle | `apps/web-astro/src/pages/games/puzzle/index.astro` |
| arcade | `apps/web-astro/src/pages/games/arcade/index.astro` |
| strategy | `apps/web-astro/src/pages/games/strategy/index.astro` |
| 3d | `apps/web-astro/src/pages/games/3d/index.astro` |

Each matching category page must have:
- The game added to the `games` array in the frontmatter
- The `numberOfItems` in the ItemList schema will auto-update (driven by `games.length`)

### 4. Homepage Structured Data
Verify `apps/web-astro/src/pages/index.astro` homepage meta description game count is updated (e.g., "Play 15+ free browser games").

## Rules

- The sitemap is auto-generated by @astrojs/sitemap — do NOT create or edit a manual sitemap file
- Verify the game page `.astro` file exists so the sitemap generator will pick it up
- If a category page doesn't exist for a genre (e.g., `sports`), note it but don't create one
- Report which category pages were verified and which need updates
